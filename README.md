板子：esp32-CH340
开发工具：vscode + platformIO + arduino
led控制引脚：P18

调试命令：onled                //开灯-常亮模式
         offled               //关灯
         setcolor 255 255 255 //设置为白色光 如果设置别的颜色可以更改后边三个参数 三个参数为RGB的值
         setlednum 18         //设置灯珠数量
         setlumi 100          //设置亮度
         onbreath             //开启呼吸模式
         onrainbow            //开启彩虹灯模式
